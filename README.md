**Desarrollado por:** Diego Alejandro Morales Rosas 
**Institución:** Universidad de Manizales

## 🌟 Visión del Proyecto
HomeEco nace de la necesidad de resolver una problemática latente en cualquier hogar: la repartición justa de las tareas. Este sistema aplica conceptos avanzados de **Programación Orientada a Objetos (POO)** para transformar el trabajo doméstico en una economía interna equilibrada y motivadora.

## 🚀 Características Técnicas (POO Avanzada)
- **Patrón State:** Gestión dinámica del ciclo de vida de las tareas.
- **Dynamic Pricing:** Algoritmo de degradación que aumenta el valor de las tareas según el tiempo de inacción.
- **Suma Cero:** Motor económico que garantiza que el esfuerzo promedio de la casa se equilibre mediante compensaciones.
- **Auditoría Cruzada:** Sistema de validación democrática para asegurar la calidad y honestidad.

## 🛠️ Estructura del Paquete
- `home_eco/contratos.py`: Interfaces y Clases Base Abstractas (ABC).
- `home_eco/motor.py`: Lógica de estados y tipos de tareas (Hogar/Urgente).
- `home_eco/economia.py`: Cerebro algorítmico y generador de reportes.
- `main_home_eco.py`: Punto de entrada y simulación del sistema.
